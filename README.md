# Relief.Ops – NGO Action Intelligence System

## Overview

Relief.Ops is an AI-powered system that converts messy NGO field data into clear, actionable decisions.

It helps organizations:

* Identify urgent needs
* Prioritize response
* Assign volunteers
* Act faster with limited resources

---

## Problem

NGOs receive data from:

* WhatsApp messages
* Field notes
* Informal reports

This data is:

* Unstructured
* Hard to analyze quickly
* Delays decision-making

---

## Solution

Relief.Ops uses AI (Google Gemini) to:

1. Extract structured insights from messy data
2. Rank needs based on urgency and scale
3. Generate realistic action plans
4. Match volunteers based on location and skills
5. Show impact and explain decisions

---

## What Makes This Stand Out

* Converts unstructured field data into real-time decisions (not just insights)
* Provides explainable AI outputs for trust and transparency
* Includes volunteer matching with contextual reasoning
* Designed for real-world NGO workflows, not just theoretical analysis
* Focuses on rapid response scenarios in Indian contexts

---

## Key Features

* AI Data Extraction (Gemini)
* Priority Scoring Engine
* Explainable AI Decisions
* Volunteer Matching System
* Impact Preview
* India-specific scenarios

---

## Tech Stack

* Frontend: React
* Backend: Node.js
* Database: MongoDB
* AI: Google Gemini API
